concepts

R basic concept

  1. indexing
    all indexing in R is from 1 not 0
    the slicing (eg 1:3): start and end are included

  2. R language is case sensitive.

  3. R operations are vector based, performs on each element of vector.

Variable naming Conventions

datatype

1. data type intro

common data type: numeric, logical, character
1. numeric represents both int and float
Note: to explicitly name integer, use suffix L, eg 1L means integer 1 1. character is string or character like python, denoted by double quote or single quote no diff 1. logical is TRUE and FALSE, you write can also T and F

2. Inspect data type

  1. mode()
    describe the data type used for storage, e.g., numeric, logical, character, etc.

  2. class()
    describe the object class of the input variable, e.g., numeric, integer, list, matrix, factor, etc.

  3. egs

    > mode(1L); class(1L) 
[1] "numeric" 
[1] "integer" 

> mode(factor(c("a","b"))); class(factor(c("a","b"))) 
[1] "numeric" 
[1] "factor" 

> mode(matrix(0,2,2)); class(matrix(0,2,2)) 
[1] "numeric" 
[1] "matrix"
